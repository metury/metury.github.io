Přednáška 1

Jazyk teorie množin

  • Jazyk teorie x ∈ Y.
  • Také se bude používat metajazyk jako například: “definovat”, “formule” a “třída”.

Symboly

  • Proměnné pro množiny X, Y, Z, x1, x2, ….
  • Binární predikátový (relační) symbol = a taky ∈ (náležení).
  • Dále také logické spojky: ¬,∧,∨,→, ← (⇐,⇒).
  • Také kvantifikátory: ∀ a ∃.
  • Samozřejmě i závorky (), [].

Formule

  • Atomické formule x = y a x ∈ y.
  1. Jsou-li φ, ψ formule, pak ¬φ, φ ∨ ψ, φ ∧ ψ, φ → ψ, φ ↔︎ ψ jsou také formule (popřípadě i uzávorkované).
  2. Je-li φ formule, pak (∀x)φ a (∃x)φ jsou také formule.
  • Každá formule pak lze dostat z atomických formulí konečně mnoha pravidly 1 a 2.

Rozšíření jazyka (zkratky)

  • x ≠ y je pro ¬(x=y).
  • x ∉ y je pro ¬(x∈y).
  • x ⊆ y je pro “x je podmnožina y” (∀u)(u∈x→u∈y).
  • x ⊂ y je pro “x je vlastní podmnožina” (x⊆y∧x≠y).

Cvičeni

Napište formulí “množina x je prázdná”.

Axiomy logiky (“jak se chovají logické symboly”)

  • Axiomy výrokové logiky např.: schéma axiomů: Jsou-li φ, ψ formule, pak

φ → (ψ→φ)

  • je axiom.
  • Axiomy predikátové logiky např.: Schéma axiomů: Jsou-li φ, ψ formule, x proměnná, která není volná ve φ, pak

(∀x)(φ→ψ) → (φ→(∀x)ψ)

  • je axiom.
  • Axiomy pro rovnost:
    • x je proměnná, pak x = x je axiom.
    • x, y, z jsou proměnné, R je relační symbol, pak

(x=y) → (∀z)(R(x,z)↔︎R(y,z))

(x=y) → (∀z)(x∈z↔︎y∈z)

(x=y) → (∀z)(z∈x↔︎z∈y)

  • Odvozovací pravidla:
    • Z φ, φ → ψ odvoď ψ.
    • Z φ′ odvoď (∀x)φ.

Axiomy teorie množin

“Jak se chová ∈.” “Jaké množiny existují.”

  • Zermelo-Fraenkelova teorie, zkráceně ZF má celkem 9 axiomů (resp. 7 axiomů a 2 schémata).
  • Pak je ještě 10.axiom výběru (AC) to pak je *ZF+AC=*ZFC.

1.Axiom existence množin

  • “Existuje množina.”

(∀x)(x=x)

2.Axiom extenzionality

  • Udává souvislost mezi  ∈  a =.
  • “Množina je určena svými prvky.”

(∀z)(z∈x↔︎z∈y) → x = y

Cvičeni

Dokažte ((x⊆y)∧(y⊆z)) → x ⊂ z.

Přednáška 2

3.Schéma axiomu vydělení

Je-li φ(x) formule, která neobsahuje volnou proměnnou z. Pak:

(∀a)(∀x)(∃z)(x ∈ z ↔︎ (x∈a∧φ(x))

je axiom.

  • “Z množiny a vybereme prvky s vlastností φ(x) a ty vytvoří novou množinu z.”
  • Díky axiomu extenzionality je taková z právě jedna.

Značení:

  • {x; x ∈ a ∧ φ(x)} je zkrácení.
  • {x ∈ a; φ(x)} “Množina všech prvků a splňující φ(x).”

Definice:

  • Průnik: a ∩ b je {x, x ∈ a ∧ x ∈ b}.
  • Rozdíl: a \ b je {x, x ∈ a ∧ x ∉ b}

Cvičení

  • Napište formulí “množina a je jednoprvková”.
  • Dokažte, že množina všech množin neexistuje.

4.Axiom dvojice

(∀a)(∀b)(∃z)(∀x)(x∈z↔︎(x=a∨x=b))

  • “(Ne)každým dvěma množinám a, b existuje množina z, která má za prvky právě a, b.”

Definice:

  • {a, b} je neuspořádaná dvojice množin a, b, jakožto dvouprvková množina s prvky a, b (pokud a ≠ b).
  • {a} znamená {a, a}, nebo-li jednoprvková množina s prvkem a.

Příklad:

Můžeme vytvořit {∅}, {{∅}}, {∅, {∅}}, …

Cvičení

Dokažte (∀z)(x∈z↔︎y∈z) → x = y.

Definice:

(a,b) je uspořádaná dvojice množin a, b. To je pak množina {{a}, {a, b}}

Poznámka:

Pro a = b je (a,b) = {{a}, {a, a}} = {{a}, {a}} = {{a}}.

Lemma

(x,y) = (u,v) ↔︎ (x=u∧y=v)

Důkaz:

  • ←
    • {x} = {u} plyne z axiomu extenzionality.
    • {x, y} = {u, v}; {{x}, {x, y}} = {{u}, {u, v}}
  • →
    • {{x}, {x, y}} = {{u}, {u, v}} to pak znamená, že {x} = {u} ∨ {x} = {u, v} kde v obou případech x = u.
    • {u, v} = {x} ∨ {u, v} = {x, y} tedy v = x ∨ v = y
    • Pokud v = x pak z x = u plyne, že v = u = x.

□

Definice:

Jsou-li a1, a2, a3, …, an množiny, definujeme uspořádanou n-tici (a1,a2,a3,…,an). Následně (a1) znamená a1 a je-li definována (a1,…,ak) pak (a1,…,ak,ak + 1) je ((a1,…,ak),ak + 1).

Lemma

(a1,a2,a3,…,an) = (b1,b2,b3,…,bn) ↔︎ (a1=b1∧…an=bn)

Důkaz:

  • Jako cvičení.

5.Axiom sumy (axiom of the union)

(∀a)(∃z)(∀x)(x∈z↔︎(∃y)(x∈y∧y∈a))

Definice:

⋃a je suma množiny a. Tzn “{x, (∃y)(x∈y∧y∈a)}”.

Pozorování

Pokud a = {b, c}, pak ⋃{b, c} = {x, x ∈ b ∨ x ∈ c}.

Definice:

b ∪ c je ⋃{b, c} sjednocení množin b, c.

Definice:

Jsou-li a1, …an množiny, definujeme neuspořádanou n-tici {a1, …an} (n-prvkovou množinu, pokud každé ai je různé) rekurzivně. Je-li definovaná {a1, …ak} pro k ≥ 2, pak {a1, …ak, ak + 1} je {a1, …ak} ∪ {ak + 1}.

6.Axiom potence (power set, potenční množina)

(∀a)(∃z)(∀x)(x∈z↔︎x⊆a)

  • “Existuje množina z jejichž prvky jsou právě podmnožiny množiny a.”

Definice:

𝒫(a) je “{x; x ⊆ a}” potenční množina [2a] množiny a (potence a).

Příklad:

𝒫(∅) = {∅} a 𝒫({∅}) = {∅, {∅}}

Cvičení

Co je 𝒫(⋃a) a jestli ⋃(𝒫(a)) = a?

7.Schéma axiomu nahrazení

  • “Obraz množiny funkcí je množina.” Je-li ψ(u,v) formule, která neobsahuje volné proměnné w, z, pak

(∀u)(∀v)(∀w)((ψ(u,v)∧ψ(u,w))→v=w) → (∀a)(∀z)(∀v)(v∈z↔︎(∃u)(u∈a∧ψ(u,v)))

je axiom.

  • “Je-li ψ funkce (částečná) určená formulí: ψ(u,v) je f(u) = v, pak obrazem a touto funkcí je opět množina (z).”
  • Také implikuje schéma vydělení: φ(u) ∧ u = v.
  • Poznámka: transfinitní rekurze, konstrukce ω + ω, Zornovo lemma, věta o dobrém uspořádání.

Přednáška 3

8.Axiom fundovanosti (foundation, regularity)

(∀a)(a≠∅→(∃x)(x∈a∧x∩a=∅))

  • “Každá množina má prvek, který je s ní disjunktní.”

Cvičení

Ukažte, že Axiom fundovanosti zakazuje existenci konečných cyklů relace ∈. Tedy množiny y takové, že y ∈ y, ale i y1, y2, …, yn takové, že y1 ∈ y2 ∈ … ∈ yn ∈ y1.

  • Díky axiomu fundovanosti lze všechny množiny vygenerovat z prázdné množiny operacemi 𝒫, ⋃.

Třídy

Definice:

φ(x) je formule a {x; φ(x)} označuje “seskupení” množin, pro které platí φ(x).

  • Pokud φ(x) je tvaru x ∈ a ∧ ψ(x), pak je to množina (axiom vydělení).
  • {x; φ(x)} je třídový term, soubor které označuje je třída určená formulí φ(x).
    • “Definovatelný soubor množin.”
  • Je-li y množina, pak y = {x; x ∈ y ∧ x = x} je třída.
    • Tedy každá množina je i třída.
  • Vlastní třída je třída, která není množinou.

Rozšíření jazyka:

  • Ve formulích na místě volných proměnných připustíme třídové termy.
  • Navíc proměnné pro třídy jsou X, Y, … (nebude možné je kvantifikovat).

Atomické proměnné

  • x = y, x ∈ y, x = X, x ∈ X, X ∈ x, X = Y, X ∈ Y
  • Plus ještě výrazy vzniklé nahrazením {x, φ(x)} za x a {y, φ(y)} za y.
  • Ostatní formule rozšířeného jazyka vznikají pomocí logických spojek (¬,∨,∧,←,→,↔︎) a kvantifikací množinových proměnných ((∀x)…(∃y)…).
  • Formule s třídovými termy bez třídových proměnných označován jako “zkrácený zápis” formule základního jazyka.
  • Formule s třídovými proměnnými označované jako “schéma formulí” základního (popř. rozšířeného) jazyka.

Eliminace třídových termů

  • x, y, z, X, Y jsou proměnné a φ(x), ψ(x) formule základního jazyka. X zastupuje {x, φ(x)} a Y zastupuje {y, φ(y)}.
  1. z ∈ X zastupuje z ∈ {x, φ(x)}.
    • “z je prvkem třídy všech množin, splňující φ(x).”
    • Nahradíme: φ(z).
  2. z = X zastupuje z = {x, φ(x)}.
    • “Množina z se rovná třídě X.”
    • Nahradíme: (∀u)(u∈z↔︎φ(u)).
  3. X ∈ Y zastupuje {x, φ(x)} ∈ {y, ψ(y)}.
    • Nahradíme: (∃u)(∀v)((v∈u↔︎φ(v))∧ψ(u)).
  4. X ∈ y zastupuje {x, φ(x)} ∈ y.
    • Nahradíme: (∃u)(∀v)((v∈u↔︎φ(v))∧u∈y).
  5. X = Y zastupuje {x, φ(x)} = {y, ψ(y)}.
    • Nahradíme: (∀u)(φ(u)↔︎ψ(v))

Meta pozorování

Formule rozšířeného jazyka určují stejné třídy jako formule základního jazyka. Příklad {x; x ∉ {z, ψ(z)}} → {x; ¬ψ(x)}.

Třídové operace

Definice:

  • A ∩ B je {x, x ∈ A ∧ x ∈ B}.
  • A ∪ B je {x, x ∈ A ∨ x ∈ B}.
  • A \ B je {x, x ∈ A ∧ x ∉ B}.
  • Pokud A = {x, φ(x)} a B = {y, ψ(y)}, pak A ∩ B = {z, φ(z) ∧ ψ(z)}.

Definice:

{x; x = x} je univerzální třída, která se značí jako V.

  • A je třída, (absolutní) doplněk A je V \ A, který se značí jako  − A.
  • A ⊆ B, A ⊂ B značí, že A je podtřídou B (popř. vlastní podtřídou).

Rozepište v základním jazyce teorie množin.

  1. ⋃A nebo-li suma třídy A je {x, (∃a)(a∈A∧x=a)}
  2. ⋂A nebo-li průnik třídy A je {x, (∀a)(a∈A→x=a)}

Přednáška 4

Relace

Definice:

  • Třída R je (binární) relace, pokud R ⊆ V × V.
  • xRy zkratka za (x,y) ∈ R.
  • n-ární relace je R ⊆ Vn.

Příklad:

  • Relace náležení E je {(x,y), x ∈ y}.
  • Relace identity Id je {(x,y), x = y}.

Definice:

Je-li X relace (libovolná třída), pak:

  • Dom(X) je {u, (∃v)(u,v) ∈ X}
  • Rng(X) je {v, (∃u)(u,v) ∈ X}
  • Je-li Y třída, pak $X \shortparallel Y (X [Y])$ je {z, (∃y)(y ∈ Y ∧ (y,z) ∈ X}.
    • Nebo-li obraz třídy Y třídou X.
  • X ↾ Y je {(y,z), y ∈ Y ∧ (y,z) ∈ X}.
    • Zúžení třídy X na třídu Y. (restrikce, parcelizace)

Lemma

Je-li x množina, Y třída, pak $Dom(x), Rng(x), x \upharpoonright Y, x \shortparallel Y$ jsou množiny.

Důkaz:

  • Vnoříme do větší množiny.
  • Platí Dom(x) ⊆ ⋃(⋃(x)).
    • Když u ∈ Dom(x) pak (∃v)(u,v) ∈ x a u ∈ {u} ∈ (u,v) ∈ x. Tedy {u} ∈ ⋃(x), tedy u ∈ ⋃(⋃(x)).
  • Podobně i pro Rng(x) ⊆ ⋃(⋃(x)).
    • v ∈ Rng(x) : (∃u)(u,v) ∈ x
    • v ∈ {u, v} ∈ (u,v) ∈ x tedy v ∈ ⋃(⋃(x)).
  • Pak už jenom $x \upharpoonright Y \subseteq x; x \shortparallel Y \subseteq Rng(x)$

□

Definice:

  • R, S jsou relace. Pak R−1 je {(u,v), (v,u) ∈ R}.
    • Nebo-li relace inverzní k R.
  • R ∘ S je {(u,v); (∃w)((u,w)∈R∧(w,v)∈S)}.
    • Nebo-li složení relací R a S.

Poznámka:

(f∘g)(x) = g(f(x))

Cvičení

  • Ověřte, že pro libovolnou relaci R je Id ∘ R = R = R ∘ Id.
  • (x,y) ∈ E ∘ E ↔︎ x ∈ ⋃y

Definice:

Relace F je zobrazení (funkce) pokud:

(∀u)(∀v)(∀w)(((u,v)∈F∧(u,w)∈F)→v=w)

  • “Pro každé v ∈ Dom(F) existuje právě jedna množina v taková, že (u,v) ∈ F.”
  • Píšeme F(u) = v.

Definice:

  • F je zobrazení třídy X do třídy Y; F : X → Y, pokud Dom(F) = X a Rng(F) ⊆ Y.
  • F je zobrazení třídy X na třídu Y; pokud navíc platí Rng(F) = Y.
  • F je prosté zobrazení pokud F−1 je zobrazení.
    • Pokud (∀v)(∀u)(∀w)((F(u)=w∧F(v)=w)→u=v).
    • “Každý prvek Rng(F) má právě jeden vzor.”

Pozorování

Pokud F je prosté zobrazení, pak F−1 je také prosté zobrazení.

Definice:

A je třída, φ je formule pak:

  • (∃x∈A)φ je zkratka za (∃x)(x∈A∧φ).
  • (∀x∈A)φ je zkratka za (∀x)(x∈A→φ).

Značení:

Obraz / vzor třídy X zobrazením F.

  • F[X] místo $F \shortparallel X$ : F[X] = {y, (∃x∈X)y = F(x)}
  • F−1[X] místo $F^{-1} \shortparallel X$ : F−1[X] = {y, (∃x∈X)x = F(y)}

Definice:

A je třída, a je množina, pak aA je {f; f : a → A}, třída všech zobrazení z a do A.
